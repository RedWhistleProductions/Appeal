Graphics Module

The Graphics module is the visual system scaffold for Appeal and Bananas. It is intended to cover windows, cameras, 2D and 3D resources, text, GUI commands, render, and update behavior through a stable Appeal-facing API.

SDL2 Backend

Appeal_SDL2_Graphics uses SDL2 for windows and rendering, Dear ImGui for GUI commands, SDL2_ttf for text, SDL2_image for image loading, and Assimp for model loading when a built Assimp library is available.

Test Script

The current test script is Resources/Appeal/Test_Graphics.Appeal. It exercises the SDL2 backend, including texture loading, text, Dear ImGui GUI commands, camera state, mesh loading, and wireframe rendering.
